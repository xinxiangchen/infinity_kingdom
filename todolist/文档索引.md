# todolist 文档索引

更新日期：2026-06-12

这个文件夹用于集中放置后续交接、待办、架构和解释性文档。后面的同学或 Codex 接手时，可以优先从这里读，不需要在仓库里到处找说明。

## 1. 本轮新增的交接文档

| 文档 | 用途 |
|---|---|
| `代码架构指南.md` | 说明当前代码架构、模块职责、角色/地图/UI/敌人/Boss/音频边界，以及后续如何解耦 |
| `策划差距与待办指南.md` | 对照策划案说明当前完成度、未完成系统、P0/P1/P2/P3 待办 |
| `Cpp重构可行性与Codex交接总结.md` | 说明 C++/GDExtension 重构是否可行、推荐迁移顺序、给下一个 Codex 的交接上下文 |
| `加点与武器升级规则待办.md` | 固定后续加点系统和武器升级系统的新规则，包括评分修正、总点数上限、武器升级触发点和数值 |
| `角色组武器升级接口约定.md` | 给角色组的武器升级实现边界和 API 约定，说明整合组只调用接口，具体表现由角色组封装 |
| `教堂商店军需库交互节点待办.md` | 固定后续教堂、商店、军需库三个地图交互节点的定位、选项和迁移方式 |
| `多结局与皇帝轮替规则待办.md` | 固定后续多结局、王子后继续轮回、家族登基、可选角色限制和小兵刷新权重规则 |
| `消耗品槽与存档扩展待办.md` | 固定后续主动消耗品槽、复杂存档结构，以及技能成长第一版暂不进入的口径 |
